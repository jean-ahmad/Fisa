Detailed Description

Class to program a pseudostate indicating the beginning of a region.

Any regions, in the machine or in "CompositeState" states, should contain an initial pseudostate. It is not necessary that a "CompositeState" state reached by a "ForkTransition" transition haves "InitialState" pseudostates in its regions.
