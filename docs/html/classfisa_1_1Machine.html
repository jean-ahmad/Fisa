Detailed Description

Class for the modeling and the simulation of automata or concurrent systems.

A machine is made of many regions. A region is where states and transitions are programmed. The regions in the machine or in "CompositeState" states are concurrent and only one transition can fired in a region. The class must be inherited and the method "build" overloaded to program the machine.
