Detailed Description

Class to manage date and time.

To work properly, input parameters constructors must match following conditions:

  • 1 <= in_month <= 12,
  • 1 <= in_day_of_year <= (365 or 366),
  • 1 <= in_day <= (28, 29, 30 or 31),
  • 0 <= in_hour <= 23,
  • 0 <= in_minute <= 59,
  • 0 <= in_second <= 59,
  • 0 <= in_usecond <= 999999.
